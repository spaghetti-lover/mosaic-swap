## Introduction
A simple swapper between USDC, Move, ETH,... coin

## Run
Create .env file in root folder with these API KEY:
```bash
NEXT_PUBLIC_PRIVATE_KEY=<YOUR_PRIVATE_KEY>
NEXT_PUBLIC_X_API_KEY=<YOUR_X_API_KEY>
```
```bash
npm install
npm run dev
```
